# Digital Brain - Second Mind App

A personal knowledge management web app built with React and Gemini AI.

## Features
- Create and manage notes with titles, content, and tags
- Visualize note connections as an interactive knowledge graph
- AI-powered chat to find insights across your notes

## Tech Stack
- React + Vite
- D3.js for knowledge graph visualization
- Google Gemini AI API
- localStorage for data persistence

## How to Run
1. Clone the repo
2. Run `npm install`
3. Create a `.env` file with your Gemini API key:
   `VITE_GEMINI_API_KEY=your-key-here`
4. Run `npm run dev`
5. Open `localhost:5173`

## Project Structure
- `src/components/` - UI components
- `src/context/` - Global state (BrainContext)
- `src/utils/` - localStorage helpers
